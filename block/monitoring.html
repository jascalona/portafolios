Virtual assistance
Ver Presentación

Virtual assistance es una interfaz desarrollada con el fin de llevar un control de asistencia "Entradas y Salidas" de tu personal, evitadno de esta forma retrasos y excusas para la puntualidad.

Este programa te permitira saber la fecha, hora y ubicacion exacta de cada uno de los usuario al momento en que realiza la carga de asistenciay y esto gracias a nuestros controladores que oblicagn al dispositivo a activar el GPS trayendo de esta forma las coordenadas en tiempo rea, (Latitude, Longitude, Amplitud). Es importante resaltar que estos datos viajaran a una base de datos donde podran mantener sus datos seguros bajo encriptasion del metodo POST.
Ahora bien, usted como usuario administrador no tendra la necesidad validar los registros en la base de datos ya que el programa cuenta con vistas por roll, esto quiere decir que cada usuario como "Operador" solo podra ver sus registros sin coordenadas, mientras usted como administrador tendra acceso a todo los registros y datos desde una terminal agradable para el usuario donde podra realizar filtros de busqueda mas precisos.
